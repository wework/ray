# Ray Docs

This is the [Ray Design System website](http://ray.wework.com) which is built using GatsbyJS.

## Structure

```
src
├── components
├── content
├── data
├── layouts
├── pages
├── styles
├── templates
```

## Development

Install dependencies

```
$ yarn
```

Run dev environment

```
$ yarn dev
```

## Build

Runing the build commands generates all the files and places them in the `public` folder.

Build

```
$ yarn build
```
