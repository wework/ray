## Modifiers

Use these modifiers with `.ray-card` class.

| Selector          | Description                                 |
| ----------------- | ------------------------------------------- |
| `.ray-card--link` | Selector for applying link and hover styles |
| `.ray-card--row`  | Selector for applying row direction         |
